## Настройка

Установите зависимости (node version >= 18)

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development - режим разработки

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## Production

Билд приложения для продакшена:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

Локальный просмотр билда:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```
